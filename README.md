# Zoom-Platform
# 🎥 Zoom Platform Clone

A frontend clone of the Zoom video conferencing platform built using HTML and CSS.
This project was built as part of a fullstack internship to practice frontend development
and GitHub team collaboration.

---

## 🔗 Live Demo

👉 [View Live Site](https://akshaya12-star.github.io/Zoom-Platform/)

---

## 📸 Pages

| Page | Description |
|------|-------------|
| 🏠 Home | Landing page with hero section and features |
| 🔐 Login | Sign in page with Google login option |
| 📊 Dashboard | User dashboard with meetings and stats |
| 🎥 Meeting Room | Live meeting UI with video grid and toolbar |

---

## 🛠️ Built With

- HTML5
- CSS3
- Font Awesome Icons
- Google Fonts (Poppins)
- GitHub Pages (Deployment)
